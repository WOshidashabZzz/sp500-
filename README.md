# 标普 500 定投提醒小程序

这是一个用于辅助查看标普 500 / SPY 定投参考信息的 Python 小程序。程序会在运行时获取可用的市场数据，结合本地缓存生成当天的参考结果，并在终端输出近期数据摘要。采用的是较为简单的策略，适合在下跌时期定投加仓，会通过pushplus直接推送到你的微信上，推荐搭配本地定时自动运行程序 

本项目主要用于个人记录和提醒，不构成任何投资建议。

## 主要功能

- 获取标普 500 / SPY 相关行情数据
- 自动维护本地缓存，减少数据源不可用时的影响
- 在终端输出当天参考信息和近期数据
- 可选接入 PushPlus，将运行结果推送到手机
- 支持通过批处理脚本在 Windows 上快速运行

## 环境要求

- Python 3.10 或更高版本
- Windows、macOS 或 Linux 均可运行

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行方式

```bash
python main.py
```

Windows 用户也可以根据自己的 Python 安装路径调整 `run_sp500.bat` 后双击运行。

## 可选推送配置

如果需要使用 PushPlus 推送，请在本机环境变量中配置：

```bash
PUSHPLUS_TOKEN=你的PushPlusToken
```

未配置时，程序仍会正常运行，只是会跳过推送。

## 文件说明

- `main.py`：主程序入口
- `requirements.txt`：项目依赖
- `run_sp500.bat`：Windows 快速运行脚本
- `sp500_cache.csv`：本地缓存文件，运行后自动生成
